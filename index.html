Toy Party.

The project is run by Helsinki Metropolia University of Applied Sciences. It gathers students to develop a full length game for kids. The game is made of 6 mini games including a memory card game, an arkanoid revival, a grabbing object game, a maze, a shooter and a line drawing puzzle.
